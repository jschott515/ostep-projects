## Prerequisites
Tested using Ubuntu 2024.04.1 (WSL)

Install the following dependencies:
- build-essential
- expect
- gcc
- gdb
- make
- qemu-system-x86

xv6 source code obtained from [mit-pdos/xv6-public](https://github.com/mit-pdos/xv6-public)
